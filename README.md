# Airline Passenger Satisfaction - Naive Bayes (89% Accuracy)

**My contribution to 4-person team project: Implemented Naive Bayes classifier with 10-fold CV achieving 89.11% accuracy**

## Team & Roles
**Team 06, DS520 Data Mining (City University of Seattle)**
- **Geraldine Marten-Ellis**: Naive Bayes implementation + evaluation
- Vuong Dang: Decision Trees
- Sumit K C: Data preprocessing  
- Nate Seon: Report writing

## What My Code Does
1. **Loads** airline passenger satisfaction dataset (103k train, 26k test records)
2. **Cleans** data - removes ID columns, converts categoricals to factors
3. **Trains** Naive Bayes with **10-fold cross-validation**
4. **Evaluates** with confusion matrix → **89.11% accuracy**
